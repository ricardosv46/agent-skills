# Agent Skills

My custom agent skills for extension of agent capabilities.

## Skills

### [Frontend Vertical-Hexagonal Architecture](skills/frontend-vertical-hexagonal/SKILL.md)
Enforces a Vertical Slice / Feature-First architecture with an internal Hexagonal Architecture (Ports and Adapters) structure using React, Next.js, and Zustand.

### [Backend NestJS Vertical-Hexagonal Architecture](skills/backend-nestjs-vertical-hexagonal/SKILL.md)
Enforces a Vertical Slice / Feature-First architecture with an internal Hexagonal Architecture (Ports and Adapters) structure and DDD using NestJS and Prisma.

## Installation

To install all skills:
```bash
npx skills add ricardosv46/agent-skills
```

To install the frontend hexagonal skill specifically:
```bash
npx skills add ricardosv46/agent-skills@frontend-vertical-hexagonal
```

To install the backend NestJS hexagonal skill specifically:
```bash
npx skills add ricardosv46/agent-skills@backend-nestjs-vertical-hexagonal
```
